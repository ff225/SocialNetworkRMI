# Social Network Distribuito

Esempio di social network distribuito utilizzando Java RMI

## Utilizzo

Avviare rmi in background:  
`rmiregistry &`

Se necessario, compilare il codice:  
`javac *.java`

Avviare il server:  
`java SocialNetworkServer`

Avviare il client:  
`java SocialNetworkClient <usn> <pwd>`
